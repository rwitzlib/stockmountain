# Support Strategy Drafts

StockMountain supports Strategy Drafts in the first version. Users edit an unpublished Strategy Draft, and publishing the draft creates an immutable Strategy Version.

**Consequences**

The Strategy editor can save work without polluting Strategy Version history. Backtest Runs and Paper Bots must use published Strategy Versions, not drafts. The first version does not support preview Backtest Runs against unpublished Strategy Drafts.
