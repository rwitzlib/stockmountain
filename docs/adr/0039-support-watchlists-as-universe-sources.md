# Support Watchlists as Universe Sources

StockMountain supports User Account-owned Watchlists as reusable Universe sources in the first version. A Watchlist can be used when creating Backtest Runs or Paper Bots.

**Consequences**

Universe configuration should be able to reference a Watchlist rather than only carrying an inline set of symbols. Watchlist ownership and symbol validation must follow the same User Account boundaries as Strategies, Backtest Runs, and Paper Bots.
