# Support Individual Accounts Only in V1

StockMountain supports individual User Accounts only in the first version. Strategies, Backtest Runs, and Paper Bots are owned by a single User Account rather than by an Organization or Workspace.

**Consequences**

The first version does not need organization membership, invitations, shared resources, or role-based workspace permissions. Clerk Organizations can be considered later if team or enterprise use cases become important.
